# Scale Ops — Auto-scaling Simulation Game

A browser-based game for the **NT BootCamp** workshop *Resource Management & Auto-scaling* (BOTNOI).
Run one month of a real system across 5 stages: fixed resources → manual → scheduled → auto-scaling → auto + graceful (12-Factor). Hold the SLO, don't run out of error budget or credits.

## Play
Open **https://lerdpisith.github.io/tn-workshop/** (no install needed — works on any modern browser, including phones).

## Files
- `index.html` — the game (self-contained, no dependencies)

## Hosting
Served as a static site via GitHub Pages from the `main` branch root.
